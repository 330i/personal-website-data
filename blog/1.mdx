I remember around the beginning of the year, I came across a blog post made by a close colleague from a previous company I interned at.
He updated the website to a clean slate.
Prioritizing function over form after realizing a professor's work speaks louder than their website.
</p>

<p style={{ textDecoration: "line-through" }}>
That actually kind of hit me.
The whole point of a personal portfolio website is to communicate who you are.
Outlandish UI elements have little to negative effect on achieving such goals.
A summarized about page communicates how you want others to think about you.
If this were to be a puzzle, you're piecing the puzzle for them.
To quote Dale Carnegie's How to Win Friends and Influence People, people "have much more faith in ideas that \[they\] discover for \[themselves\] than in ideas handed to \[them\] on a silver platter."
So for the new year, I want to change this website to serve as a place for discovery.
</p>

<p style={{ textDecoration: "line-through" }}>
It's quite fitting because I've always wanted a variable-effort repository for my life and project updates.
I don't like using short-form, centralized, far-reaching social media (I've scrolled through Instagram for 3 months in 10th grade and never looked back, I don't think I'll ever get to use Snapchat and TikTok), so this seems like a perfect opportunity.
</p>

Starting a new thing this year.
I've always wanted to have a repository for my life and project updates but hate using social media.
Since I know no one looks at this website, I think it's a great place to start a little recession-era blog.
It'll be great practice for researching and explaining things I don't quite understand yet.

(Ignore the fact that I wrote this months before the new version of the website went live.)
